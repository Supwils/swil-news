## 二、大模型 · 研究与 Agent

### 3. 用编码 Agent 自动搜寻「测试时缩放」控制算法：更少算力换同等或更好精度（研究速递）
**摘要：** THE DECODER 于 5 月 24 日报道一项研究思路：团队在模拟环境中让 Claude Code 多轮迭代搜索更优的控制算法，用于改进大模型在测试阶段的推理/缩放策略。报道援引论文称，在部分数学基准上，新算法在精度—算力权衡上优于既有代表性方法；在一种与「自洽多数投票」类基线对比的设置中，报道给出约七成级别的 token 使用下降幅度（具体实验设定以论文为准）。作者称整套自动发现流程成本与耗时相对较低，并强调这类「由 Agent 写算法」的路径可能改变未来缩放策略设计方式。
**链接：**
- [THE DECODER — Researchers let Claude Code discover AI scaling algorithms](https://the-decoder.com/researchers-let-claude-code-discover-ai-scaling-algorithms-that-humans-probably-wouldnt-have-designed/)
**简评：** 当「缩放」从手工技巧变成可搜索的程序空间，竞争焦点会从单点模型参数转向工具链、评测环境与自动实验预算——也更接近芯片 EDA 领域早已发生的「人机协同优化」。

## 三、内容溯源 · 平台安全

### 4. 深伪压力下的「可验证来源」竞赛：OpenAI 公共核验页 + Google 扩大 SynthID / C2PA 核验入口
**摘要：** OpenAI 提供面向公众的图像上传核验流程（`openai.com/verify`），用于检测图像是否带有与其生成工具相关的 C2PA 与 SynthID 等溯源信号；官方说明强调工具仅用于判断是否由其工具链路生成，并不等价于「内容真实或未篡改」。Google 在 5 月 19 日的官方博文中宣布，将把 SynthID 以及 C2PA Content Credentials 相关核验能力扩展到搜索等产品路径，并提到 Gemini 应用内相关能力已被大量使用；同日博文亦介绍面向企业的云侧检测 API 路线与生态合作方向。
**链接：**
- [OpenAI — Verify OpenAI-generated images](https://openai.com/research/verify/)
- [Google — Identifying AI-generated media online](https://blog.google/innovation-and-ai/products/identifying-ai-generated-media-online/)（发布日期：2026-05-19）
**简评：** 平台侧互认 C2PA + 水印，是在「生成即泛滥」世界里把争议从「是不是 AI」部分推回「是谁生成、是否被继续篡改」；但跨厂商覆盖不均时，核验工具也可能被误读为全局真相仲裁器。

### 10. CMR-CLIP：以心血管磁共振（CMR）「视频 + 影像报告」对齐学习，减少对手工标注依赖（Nature Communications）
**摘要：** 卡内基梅隆大学与克利夫兰诊所合作团队在 *Nature Communications* 发表 CMR-CLIP：将心血管磁共振（CMR）检查视作随时间变化的心脏影像序列，并与对应放射学报告文本做对比语言—图像预训练，从而在多种临床相关任务上取得较强表现。克利夫兰诊所新闻稿称模型在超过 1.3 万例去标识化真实患者研究上训练，并在测试中相较通用 AI 模型在部分指标上提升超过 35%；论文亦给出多种疾病分类准确率等量化结果。代码与权重以公开仓库形式提供。
**链接：**
- [Nature Communications — CMR-CLIP paper](https://www.nature.com/articles/s41467-026-73022-2)
- [Cleveland Clinic Newsroom — CMU and Cleveland Clinic cardiac MRI AI](https://newsroom.clevelandclinic.org/2026/05/21/carnegie-mellon-university-and-cleveland-clinic-develop-ai-system-to-interpret-cardiac-mri-scans-with-enhanced-accuracy)
**简评：** 医疗影像领域「弱标注/自然语言监督」路线继续证明：与其追逐通用多模态榜单，不如把临床书写习惯本身变成可学习的结构先验。

**总体定性：** 今天更像「**治理—国安—供应链三角问题被同时点亮**」的一日：政策信号剧烈摇摆、情报与商业模型合同化、算力与实体 AI 基础设施继续抢跑。
