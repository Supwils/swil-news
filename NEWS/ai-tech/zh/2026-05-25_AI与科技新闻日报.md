### 3. 微软被曝收紧内部 Claude Code 许可：Experiences + Devices 团队转向 GitHub Copilot CLI，6 月底前完成迁移（组织策略）
**摘要：** 据 *The Verge* 等多家媒体援引知情人士，微软正取消面向大量内部员工的 **Anthropic Claude Code** 直接许可，负责 Windows、Microsoft 365、Outlook、Teams 与 Surface 的 **Experiences + Devices** 组织被要求逐步迁移至 **GitHub Copilot CLI**，目标在 **2026 年 6 月 30 日**前完成切换。报道指该工具过去约半年在工程师中「过于受欢迎」，推高了 token 与相关运营成本；公司对外合作层面仍强调与 Anthropic 在 Azure / Foundry 等渠道的大客户与模型接入关系未因内部工具调整而终止。*The Next Web* 等评论将时间线与财年末预算周期并置解读。
**链接：**
- [The Verge — Microsoft starts canceling Claude Code licenses](https://www.theverge.com/tech/930447/microsoft-claude-code-discontinued-notepad)
- [The Next Web — Microsoft’s quiet Claude Code retreat and the real cost of enterprise AI](https://thenextweb.com/news/microsoft-claude-code-retreat-ai-cost)
